Detailed Description

This class provides functionality for the operation of up to 4 DC Motors driven by the PCA9685 PWM-Chip and the TB6612FNG Motor-Bridge as found on the Adafruit Arduino Motorshield V2. This class is implemented as a singleton since this class shall solely be used for the chassis motors.
